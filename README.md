# Gym Workout Progress Tracking

A fully feature Flutter application to log and track your gym workouts and fitness progress.

## Overview

This Gym Workout Tracking app allows users to create workouts, add exercises, log sets (reps & weight), and track their progress over time. Built with Flutter, it offers a smooth, cross-platform experience. It utilizes `provider` for state management, `sqflite` for local persistent storage, and `fl_chart` to visualize your gains.

## Features

*   **Custom Workouts:** Create and manage customized workouts.
*   **Exercise Logging:** Add various exercises to your workouts and record reps and weights for every set.
*   **Progress Tracking & Charts:** Visualize your strength improvements with easy-to-read charts.
*   **Local Storage:** Completely offline experience. All data is saved directly on your device via SQLite.

## Technologies Used

*   [Flutter](https://flutter.dev/) - Framework for the UI
*   [Provider](https://pub.dev/packages/provider) - State Management
*   [sqflite](https://pub.dev/packages/sqflite) - Local Database
*   [fl_chart](https://pub.dev/packages/fl_chart) - Data Visualization

## Getting Started

### Prerequisites

*   Install [Flutter](https://docs.flutter.dev/get-started/install)
*   An IDE (VS Code, Android Studio, IntelliJ, etc.)

### Installation

1.  **Clone the repository** (if applicable) or download the source code:
    ```bash
    git clone https://github.com/AyoubAlswida/Gym-Workout-Progress-Tracking.git
    cd Gym-Workout-Progress-Tracking
    ```

2.  **Get dependencies:**
    ```bash
    flutter pub get
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```
